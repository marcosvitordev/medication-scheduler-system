# Sistema AT - Frontend

Frontend operacional do Sistema AT para cadastro de pacientes, rotina, prescrição e calendário posológico.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- shadcn/ui style components
- TanStack Query
- React Hook Form
- Zod
- date-fns
- lucide-react

## Configuração

Crie um arquivo `.env.local`:

```env
BACKEND_API_URL=http://localhost:3000/api
```

## Rodando

```bash
npm install
npm run dev
```

Abra `http://localhost:3001` se a porta 3000 estiver ocupada pelo backend.
